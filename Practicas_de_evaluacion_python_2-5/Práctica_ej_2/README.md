# Ejercicio 2

Este ejercicio pide al usuario un número binario y lo convierte a decimal.  
La función principal es `esBinario(strbinario)` que valida si una cadena es binaria.

## Estructura

- `app/main.py`: Programa principal.
- `app/modulo1/funciones.py`: Función `esBinario`.
- `tests/test_modulo1.py`: Pruebas unitarias con `unittest`.
