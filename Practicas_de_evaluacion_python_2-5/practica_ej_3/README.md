# Ejercicio 3

Este ejercicio solicita al usuario un número entre 1 y 20 y comprueba si está dentro de una lista específica.  
Se utilizan dos funciones:

- `estaEnRango(valor, minimo, maximo)`
- `estaEnLista(valor, lista)`

## Estructura

- `app/main.py`: Programa principal.
- `app/modulo1/lista.py`: Funciones reutilizables.
- `tests/test_modulo1.py`: Pruebas unitarias con `unittest`.

## Ejecutar el programa

```bash
python app/main.py
